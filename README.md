# GymTracker

A minimal workout viewer built with React and Material Design 3. Installable as a PWA.

## Features

- **Push/Pull/Legs/Recall** workout split
- **Adjustable weights** with localStorage persistence
- **Rest timer** (default 2:30, customizable)
- **Add/remove exercises** on the fly
- **Collapsible 4-week periodization** (optional)
- **Material You** dynamic colors on Android 14+
- **Dark/Light mode**
- **PWA** - installable, works offline

## Install

**Android (Chrome):**
Menu (⋮) → "Add to Home Screen"

**iOS (Safari):**
Share (↑) → "Add to Home Screen"

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
